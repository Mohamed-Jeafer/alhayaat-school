# Backlog (markdown tasks)

This folder holds **task**, **milestone**, and **doc** markdown files (for example `tasks/task-004 … .md`). There is **no** single `backlog.md` file at the repository root.

For **Backlog.md** workflow hints, see **`AGENTS.md`** in the repo root: prefer **MCP** in Cursor when connected; otherwise use the **Backlog.md CLI** (same workflow, with guardrails against bad task edits, creates, or closes—avoid ad-hoc raw markdown for lifecycle changes).

**Production hosting** (DNS, Azure, cPanel redirects): **`docs/PRODUCTION-HOSTING.md`** and **`cpanel_zone_editor.md`**.
